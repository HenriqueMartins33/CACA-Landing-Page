A identidade visual foca-se no azul e branco, cores adequadas ao contexto clínico e académico. Utilizou-se a fonte Roboto para garantir legibilidade. O estilo geral é moderno e minimalista.
O desenvolvimento progrediu de wireframe simples para um mockup de alta fidelidade. A implementação final seguiu este guia, registando-se melhorias ou alterações nos alinhamentos e posições de logos, visando um aspeto mais polido e visualmente agradável.
Adotaram-se etiquetas HTML5 para organizar a hierarquia da informação. Foram incluídos atributos alt em todas as imagens e labels associadas aos inputs do formulário para garantir a acessibilidade. A responsividade é assegurada por Media Queries que reorganizam o layout de colunas para uma estrutura vertical de coluna unica em dispositivos móveis.
